This app adds Dawson Creek Transit System (BC Transit) buses information to MonTransit.

This app provides the buses schedule and news from @BCTransit on Twitter.

Dawson Creek Transit System buses serve the Dawson Creek in British Columbia, Canada.

Once this application is installed, the MonTransit app will display buses information (schedule...).

This application only has a temporary icon: download the MonTransit app (free) in the "More ..." section bellow or by following this Google Play link https://goo.gl/pCk5mV

You can install this application on the SD card but it is not recommended.

The information comes from the GTFS file provided by BC Transit.
https://www.bctransit.com/open-data

This application is free and open-source:
https://github.com/mtransitapps/ca-dawson-creek-transit-system-bus-android

This app is not related with BC Transit and Dawson Creek Transit System.

Permissions:
- Other: required to read news from Twitter
